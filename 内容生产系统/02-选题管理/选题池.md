# 选题池

> 本地 Markdown 格式管理选题，便于版本控制和 AI 检索

---

## 选题记录模板

```markdown
## 选题记录 - {日期}

**标题：** {选题标题}
**状态：** 🆕 新选题 / 📋 待深化 / ✍️ 写稿中 / 🔄 修改中 / ✅ 待发布 / 🚀 已发布 / 📊 复盘中
**标签：** {标签1}, {标签2}, {标签3}
**优先级：** 🔴 高 / 🟡 中 / 🟢 低
**预计时长：** {15s/30s/60s/3min}

### 内容概要
**摘要：** {一句话描述选题}
**核心观点：** {核心要表达什么}
**目标受众：** {给谁看的}

### 背景信息
**灵感来源：** {从哪里获得灵感}
**参考案例：** {类似爆款或参考}
**可延伸方向：** {还能做什么相关内容}

### 执行记录
- 创建时间：{日期}
- 计划发布：{日期}
- 实际发布：{日期}
- 关联文稿：{文稿链接}

### 数据表现（发布后填写）
- 播放量：{数字}
- 点赞数：{数字}
- 评论数：{数字}
- 收藏数：{数字}
- 复盘结论：{经验总结}
```

## 使用说明

1. **新增选题**：使用 `/记录选题` 命令，AI 会自动按模板添加
2. **状态更新**：直接编辑状态标签
3. **检索选题**：AI 可以全文检索所有选题内容
4. **数据关联**：发布后关联数据表现，形成完整闭环
