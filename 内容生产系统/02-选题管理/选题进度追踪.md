# 选题进度追踪

> 追踪每个选题从想法到发布的完整流程

---

## 状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| 新选题 | 🆕 | 刚记录，还没处理 |
| 待深化 | 📋 | 决定做，还在准备素材 |
| 写稿中 | ✍️ | 正在生成文稿 |
| 修改中 | 🔄 | 文稿修改完善 |
| 待发布 | ✅ | 定稿了，等发布 |
| 已发布 | 🚀 | 发布了，记录数据中 |
| 复盘中 | 📊 | 数据回来了，做复盘 |
| 已归档 | 📦 | 复盘完成，归档保存 |
