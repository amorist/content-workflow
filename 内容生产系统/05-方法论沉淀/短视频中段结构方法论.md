## 中段六步结构

短视频中段（正文部分）遵循 **6步递进结构**，从产品出现到最终收口，层层推进建立信任。

```
产品出现 → 场景 → 感受 → 卖点 → 案例 → 收口
```

---

## 第一步：产品出现

**目的：** 自然引入产品，不突兀

**核心话术：**

| 话术类型 | 示例 |
|----------|------|
| 自用推荐 | "这个就是我一直在用的" |
| 强烈推荐 | "这个东西你一定要备一个" |
| 关键提醒 | "这个真的很关键" |
| 秘密分享 | "偷偷告诉你一个好东西" |
| 必备神器 | "这是我包里必备的" |

**使用技巧：**
- 用"这个"指代，不直接说产品名
- 强调"一直在用"增加可信度
- 语气自然，像朋友推荐

---

## 第二步：场景

**目的：** 让用户想象使用画面

**常见场景：**

| 场景类型 | 示例 |
|----------|------|
| 随身携带 | "放在包里"、"随身携带" |
| 家居摆放 | "挂在衣柜"、"摆在床头"、"放在玄关" |
| 出行使用 | "放在车里"、"出门时带上" |
| 办公场景 | "放在办公桌上"、"电脑前放一个" |
| 特定时刻 | "出门前喷一下"、"睡前用" |

**使用技巧：**
- 场景要具体可想象
- 与目标人群生活相关
- 多给几个场景选择

---

## 第三步：感受

**目的：** 描述使用后的感官体验

**感受类型：**

| 感官 | 描述词汇 |
|------|----------|
| **嗅觉** | 闻着很舒服、味道很清新、香气持久 |
| **触觉** | 手感很好、很柔软、质地细腻 |
| **视觉** | 看起来很精致、颜值很高、包装好看 |
| **整体状态** | 整个人很轻松、状态很干净、感觉很顺 |
| **心理感受** | 心里很踏实、感觉更自信、心情变好了 |

**话术示例：**
- "闻着很舒服，整个人都很放松"
- "状态变得很干净，感觉特别顺"
- "用起来很安心，心里踏实"

---

## 第四步：卖点

**目的：** 突出产品核心优势

**卖点类型：**

| 卖点维度 | 示例 |
|----------|------|
| **便利性** | 很方便不占地方、一贴就可以用、随身携带 |
| **持久性** | 味道很持久、能用很久、效果持续 |
| **性价比** | 很划算、一杯奶茶钱、能用很久 |
| **品质感** | 用料很好、做工精致、品质有保障 |
| **独特性** | 市面上很少见、独特配方、专属设计 |

**话术示例：**
- "很方便不占地方，放包里完全不占空间"
- "味道很持久，一个能用好几个月"
- "一贴就可以用，特别省事"

---

## 第五步：案例

**目的：** 用他人体验建立信任

**案例类型：**

| 案例来源 | 话术示例 |
|----------|----------|
| **粉丝反馈** | "很多粉丝用了反馈很好" |
| **老粉回购** | "老粉都在回购" |
| **报喜案例** | "用过都来报喜" |
| **身边朋友** | "我身边的朋友都在用" |
| **自己体验** | "我自己用了X个月，真的有效" |

**使用技巧：**
- 用"很多"、"都"体现普遍性
- 强调"回购"体现产品好
- 具体数字更有说服力

---

## 第六步：收口

**目的：** 促进行动，引导下单

**收口类型：**

| 收口方式 | 话术示例 | 适用场景 |
|----------|----------|----------|
| **紧迫感** | "赶紧用起来"、"不要等没货" | 限量/限时 |
| **行动指令** | "现在就带回去"、"直接拍" | 直播间 |
| **利益引导** | "现在拍有优惠"、"今天下单送XX" | 促销活动 |
| **从众心理** | "大家都在用"、"很多姐妹已经拍了" | 热销款 |
| **简单直接** | "链接在左下角"、"点下方小黄车" | 短视频带货 |

## 完整中段示例

### 示例1：香包产品

> **产品出现：** 这个就是我一直在用的香包
> **场景：** 放在包里，随身携带
> **感受：** 闻着很舒服，整个人都很放松
> **卖点：** 很方便不占地方，味道还很持久
> **案例：** 很多粉丝用了反馈很好，老粉都在回购
> **收口：** 赶紧用起来，不要等没货了

### 示例2：眼部喷雾

> **产品出现：** 这个东西你一定要备一个
> **场景：** 出门前喷一下，放在包里随时用
> **感受：** 眼睛很舒服，状态变得很干净
> **卖点：** 一喷就可以，特别方便
> **案例：** 用过都来报喜，说眼睛不干涩了
> **收口：** 现在就带回去，链接在左下角

### 示例3：收纳盒

> **产品出现：** 这个真的很关键
> **场景：** 放在玄关、摆在床头
> **感受：** 整个人很轻松，家里很整洁
> **卖点：** 很能装还不占地方
> **案例：** 很多粉丝说家里整洁了，心情都好了
> **收口：** 赶紧拍，今天下单还有优惠

## 中段结构检查清单

生成中段内容后，检查以下要素：

- [ ] **产品出现** - 是否自然引入，不突兀？
- [ ] **场景** - 是否有具体使用场景？
- [ ] **感受** - 是否描述了感官/心理感受？
- [ ] **卖点** - 是否突出了核心优势？
- [ ] **案例** - 是否有他人使用反馈？
- [ ] **收口** - 是否有明确的行动引导？

## 中段与开头公式的配合

中段结构配合「爆款文案核心公式」使用：

```
【开头】节点 + 人群 + 痛点 + 解释 + 动作
  ↓
【中段】产品 → 场景 → 感受 → 卖点 → 案例 → 收口
```

**配合示例：**

> **开头：** "立秋前，所有女生一定要把包整理一下，因为包是米库"
> **中段：** "这个香包就是我一直在用的，放在包里随身携带，闻着很舒服，很方便不占地方，很多粉丝用了反馈很好，赶紧用起来"
