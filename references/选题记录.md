# 选题记录规范

## 选题记录模板

```markdown
## {选题标题}
- 时间：{YYYY-MM-DD HH:mm}
- 标签：#AI #{具体领域} #{内容类型}
- 状态：🆕新选题
- 摘要：{一句话说清楚这个选题的核心观点}

### 背景/灵感来源
{为什么会想到这个选题？}

### 核心观点
{你想传达的最重要的1-2个观点}

### 预计受众
{谁会看这条内容？他们的痛点是什么？}

### 可延伸方向
- 方向A：{角度A}
- 方向B：{角度B}
```

## 标签体系

### 按领域
- #AI #AI工具 #AI写作 #AI图像 #AI视频 #AI Agent #RAG #大模型

### 按内容类型
- #干货 #观点 #教程 #测评 #故事 #热点 #盘点

### 按情绪
- #共鸣型 #激励型 #启发型 #实用型

## 选题质量判断

### 好选题的特征
1. 有明确的受众痛点或好奇心
2. 有独特的观点或角度（不是泛泛而谈）
3. 在自己能力范围内能讲清楚
4. 有素材支撑（案例、数据、故事）

### 自检问题
- [ ] 这个选题解决什么问题？
- [ ] 用户为什么要看？
- [ ] 我的差异化在哪里？
- [ ] 素材库有没有相关内容可以复用？
