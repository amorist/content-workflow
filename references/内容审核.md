# 内容审核工作流

## 触发条件

- 「审核内容」
- 「检查一下」
- 「看看有没有问题」
- 「内容审核」
- 用户完成文稿后

## 执行步骤

### Step 1: 运行前置检查

```bash
bash /Users/amor/Documents/智伴同行/AI知识付费/content-workflow/.content-workflow/preamble.sh
```

### Step 2: 读取待审核内容

如果用户未指定文件，询问：

> 请提供要审核的文稿路径，或粘贴文稿内容。

### Step 3: 四重审核

依次执行4个审核角色，**每个角色独立输出审核结果**：

---

## 审核员 1: 合规审核员

**职责：** 检查平台规则、法律法规、敏感内容

### 检查清单

#### 平台规则
- [ ] 是否涉及虚假宣传？
- [ ] 是否使用极限词（最、第一、绝对等）？
- [ ] 是否涉及医疗/金融/教育等敏感领域资质？
- [ ] 是否有违规引流（二维码、微信号等）？
- [ ] 是否涉及低俗/暴力/歧视内容？

#### 法律法规
- [ ] 是否侵犯他人肖像权/名誉权？
- [ ] 是否使用未经授权的音乐/图片？
- [ ] 是否涉及虚假广告法违规？

#### 内容安全
- [ ] 是否涉及政治敏感话题？
- [ ] 是否传播伪科学？
- [ ] 是否制造焦虑/恐慌？

### 输出格式

```markdown
### 合规审核报告

**审核结果：** ✅ 通过 / ⚠️ 需关注 / ❌ 需修改

**风险项：**
1. **[{等级}]** {问题描述}
   - 位置：{具体位置}
   - 法规依据：{相关规则}
   - 修改建议：{具体建议}

**通过项：**
- {检查通过的项}
```

## 审核员 2: 数据预测师

**职责：** 基于历史数据预测内容表现

### 分析维度

1. **开头吸引力预测**
   - 前3秒是否足够抓人？
   - 与历史爆款开头对比
   - 预估3秒完播率

2. **互动潜力预测**
   - 是否有引发评论的点？
   - 是否有引发转发的价值？
   - 预估互动率

3. **完播率预测**
   - 内容节奏是否合理？
   - 是否有掉风险点？
   - 预估完播率

4. **转化潜力预测**（带货内容）
   - 痛点挖掘是否到位？
   - 产品卖点是否清晰？
   - 转化路径是否顺畅？

### 输出格式

```markdown
### 数据预测报告

**整体评级：** ⭐⭐⭐⭐⭐ (5星制)

**关键指标预测：**
| 指标 | 预测值 | 置信度 | 说明 |
|-----|-------|-------|------|
| 3秒完播率 | X% | 高/中/低 | {分析} |
| 整体完播率 | X% | 高/中/低 | {分析} |
| 点赞率 | X% | 高/中/低 | {分析} |
| 评论率 | X% | 高/中/低 | {分析} |
| 转发率 | X% | 高/中/低 | {分析} |

**优势分析：**
1. {内容亮点}

**风险点：**
1. {可能的问题}

**优化建议：**
1. {具体建议}
```

## 审核员 3: 文案优化师

**职责：** 改进表达、增强吸引力、提升转化

### 优化维度

1. **开头优化**
   - 前5秒是否足够抓人？
   - 是否有更好的钩子方式？
   - 与爆款开头对比

2. **表达优化**
   - 是否有生硬/书面语？
   - 是否有更好的口语化表达？
   - 节奏是否流畅？

3. **结构优化**
   - 逻辑是否清晰？
   - 是否有冗余内容？
   - 信息密度是否合适？

4. **转化优化**（带货内容）
   - CTA是否明确？
   - 紧迫感是否足够？
   - 信任背书是否充分？

### 输出格式

```markdown
### 文案优化报告

**优化建议：**

#### 开头优化
**原文：**
```
{原文}
```
**建议改为：**
```
{优化后}
```
**理由：** {为什么这样改更好}

#### 表达优化
**原文：**
```
{原文}
```
**建议改为：**
```
{优化后}
```
**理由：** {为什么这样改更好}

#### 结构优化
{结构调整建议}

#### 其他建议
{其他改进点}
```

## 审核员 4: 平台适配师

**职责：** 优化封面、标题、标签、发布时间

### 优化维度

1. **封面优化**
   - 封面文案是否吸引人？
   - 字体/颜色建议
   - 画面构图建议

2. **标题优化**
   - 是否有关键词？
   - 是否符合平台算法偏好？
   - 是否有更好的表达方式？

3. **标签优化**
   - 标签是否精准？
   - 是否有热门标签？
   - 标签组合策略

4. **发布策略**
   - 最佳发布时间
   - 发布频率建议
   - 互动策略

### 输出格式

```markdown
### 平台适配报告

**平台：** {抖音/小红书/视频号等}

#### 封面优化
**建议文案：** {封面文字}
**视觉建议：** {颜色/字体/布局}

#### 标题优化
**原标题：** {原标题}
**优化建议：**
1. {建议1}
2. {建议2}
3. {建议3}

#### 标签策略
**推荐标签：**
- 核心标签：{精准标签}
- 热门标签：{流量标签}
- 长尾标签：{细分标签}

#### 发布策略
**最佳时间：** {时间段}
**发布技巧：** {具体建议}
```

## 审核标准

### 合规审核
- ❌ 任何红线问题必须修改
- ⚠️ 灰色地带建议修改
- ✅ 无风险直接通过

### 数据预测
- ⭐⭐⭐⭐⭐ 爆款潜力，建议发布
- ⭐⭐⭐⭐ 优质内容，可发布
- ⭐⭐⭐ 普通内容，建议优化
- ⭐⭐及以下 风险较高，建议大改

## 特殊情况处理

**如果内容涉及敏感领域：**
- 医疗/健康：额外检查资质声明
- 金融/投资：额外检查风险提示
- 教育/培训：额外检查效果承诺

**如果内容数据预测较低：**
- 分析原因（选题/开头/结构）
- 给出具体改进方向
- 提供参考案例
