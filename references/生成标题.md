# 短视频标题方法论

## 七种高点击率标题类型

### 1. 痛点+方案型
**公式：** {描述痛点}，{解决方案}
**示例：**
- 「写作没灵感？这个AI工具3秒生成初稿」
- 「不会写Prompt？记住这个万能公式」
- 「ChatGPT总答非所问，因为你问的方式不对」

### 2. 好奇+悬念型
**公式：** {制造悬念/反差} + {不完全揭示答案}
**示例：**
- 「AI不会取代你，但会用AI的人会」
- 「我把ChatGPT接入了我的工作流，从此不用上班」
- 「用了30天AI，我发现一个秘密」

### 3. 数字+利益型
**公式：** {具体数字}个{主题}，{收益}
**示例：**
- 「5个让你效率提升10倍的AI工具」
- 「3个Prompt技巧，让AI输出质量翻倍」
- 「学会这4步，AI写作不再是人工智障」

### 4. 身份+共鸣型
**公式：** {特定人群}必看/{做过某事}的人
**示例：**
- 「做自媒体的人，2024年一定要学会用AI」
- 「天天加班的你，需要这个AI工作流」
- 「独立开发者都在用的AI提效方案」

### 5. 成果+展示型
**公式：** {展示成果}，附{方法/工具}
**示例：**
- 「我用AI副业月入3万，用的是这个组合」
- 「30天AI学习成果：从小白到熟练掌握10种工具」
- 「这是我用AI写的第一本书，销量破万」

### 6. 对比/反差型
**公式：** {A} vs {B}，{选择}
**示例：**
- 「Claude vs ChatGPT，选哪个？一文说清楚」
- 「AI写作 vs 人工写作，真实对比来了」
- 「国产AI vs GPT-4，我为什么选择它」

### 7. 教程+步骤型
**公式：** {目标}，只需{步骤数}步
**示例：**
- 「AI做账号规划，只需3步」
- 「零基础用AI做副业，全流程详解」
- 「5分钟学会用AI写爆款文案」

## 标题生成流程

1. 读取文稿核心内容（开场钩子 + 3个要点 + 结尾）
2. 提取核心主题、关键观点、可量化的点
3. 按上述7种类型各生成1-2个标题
4. 选择5个最有潜力的展示给用户

## 标题输出格式

```markdown
## 标题选项

### 痛点+方案型
1. 「{标题}」— 痛点精准，解决方案明确
2. 「{标题}」

### 好奇+悬念型
3. 「{标题}」— 制造悬念，引发点击
4. 「{标题}」

### 数字+利益型
5. 「{标题}」— 具体数字，可信度高

---

## 推荐

综合考虑，优先推荐 **第X个标题**：
- 理由：{分析选择理由}
- 类型：{所属类型}
- 适合平台：{视频号/抖音/小红书}
```

## 标题自检清单

- [ ] 是否在3秒内让人想点击？
- [ ] 是否明确说明了内容价值？
- [ ] 是否匹配视频内容（不要标题党）？
- [ ] 是否适合目标平台的用户习惯？
- [ ] 是否有关键词利于搜索？

## 平台差异

| 平台 | 标题风格 |
|------|---------|
| 视频号 | 偏生活化、有温度、适合朋友圈转发 |
| 抖音 | 偏冲击感、数字多、情绪强烈 |
| 小红书 | 偏实用、干货足、有获得感 |
